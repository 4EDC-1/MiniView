# MiniView

MiniView is a lightweight, always-on-top developer window for testing HTML content with **live hot-reload**. 

## Features

- Always-on-top small window
- Automatic live reload when content changes
- Supports English and UTF-8 content
- Easy to integrate into any project

git clone https://github.com/<4EDC-1>/MiniView.git
